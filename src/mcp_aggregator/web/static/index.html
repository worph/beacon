Setup

Add Beacon to your LLM client to access all discovered tools through a single MCP connection.

Claude Code (CLI)

claude mcp add beacon --transport http http://localhost:9099/mcp

MCP Settings (JSON)

{
  "mcpServers": {
    "beacon": {
      "type": "streamableHttp",
      "url": "http://localhost:9099/mcp"
    }
  }
}

How It Works

  1. Beacon broadcasts a UDP discovery packet on the mcp-net Docker network every 30s
  2. MCP servers with a discovery responder reply with their name, description, and tool list
  3. Beacon registers all tools with namespacing: {server}__{tool}
  4. Your LLM client connects once to Beacon and gets access to every tool from every server
